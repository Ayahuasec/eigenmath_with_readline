#

Eigenmath uses UTF-8 encoding to display Greek letters and other symbols.

To build and run

```
make
./eigenmath
```

Press ctrl-C to exit.

To run a script

```
./eigenmath scriptfilename
```

To run test scripts

```
./eigenmath test/selftest1
./eigenmath test/selftest2
```
