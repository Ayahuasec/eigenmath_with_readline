#

To build and run

```
make
./eigenmath
```

Press ctrl-C to exit.

Eigenmath uses UTF-8 encoding. The terminal window should have UTF-8 enabled.

To run a script

```
./eigenmath scriptfilename
```

To run test scripts

```
./eigenmath test/selftest1
./eigenmath test/selftest2
```
